The Code

What did you do and why?

To caclulate the frequency of each word within the song I decided it would be best to use a hashmap. Hashmaps allowed me to pair a term with an integer. The added benefits is the ability to change the integer as you go along. With this functionality a lot can be accomplished, one example being creating polls -- where a choice is connected to a value that can increase.
